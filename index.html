This is a list of commands to remember for coding with python:

Lets begin...

The "print()" command is a extremely basic command, used to make words appear on the screen. You can either print text or the values of a variable or list. The "input()" command is used to get user input. It is used to ask the player something, and can then be used to store the answer in a variable. Variables are tricky. They go "name = value". In the example "x = 5 print(x)", the program would print "5". Lists are a very important part of programming. They are used to store multiple values in one variable. They are made with "list = [1, 2, 3, 4, 5]".

By importing "turtle", you can create drawings using code. You can rename turtle by doning "pen = turtle.Turtle()". When you want to run the turtle command to draw a line, you would have to do "pen.forward(50)". You can also change the color of the pen by doing "pen.color("red")".

By importing "random", you can do stuff like "random.randint(1, 10)" to create a random number 1-10. An "if/else" statement allows you to do something only if something else is true. You can use "elif" for additional if statements. End it all off with a "else". An "while" loop is like an "if" statement, but the command repeats until the condition of the "while" loop is false.

The "user_input" command allows you to read a user's input to do things, like detect words in it.

The datetime commands you get by importing "datetime" allow you to do many things, like get ing the minute, hour, day, month, and year.

The "<p></p>" command allows you to make a paragraph of text. For example, the code "<p>Hello world!</p>" would read "Hello world!"
